## Ideas Notebook
This app lets you store your idea, modify and delete them in the future and also sort them with respect to the title and created date.

### Installing and running the app

Clone the app and install the dependecies using `yarn`, once the depedencies are installed run `yarn start` to start the proejct, it will automatically open up the browsers.

### Major features of the App
- Add new ideas
- Delete/Update the existing ideas
- Sort the idea

### Enviroment
- React version 16.12.0
- Node version 12.4.0

### Tools/Features
- Typescript
- ES and TS linting
- Prettier
- Husky hooks with lint-staged
